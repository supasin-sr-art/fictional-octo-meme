BLACKWOLF Web V2.5 Run Control

- เพิ่มปุ่ม "หยุดและลบ" ในแต่ละแถวของหน้าประวัติการรัน
- หาก Run กำลังทำงาน ระบบจะหยุดเฉพาะ Process Tree ของ Run ที่เลือก
- ลบเฉพาะโฟลเดอร์ Input/Output/Log ที่ระบบคัดลอกไว้สำหรับ Run นั้น
- ไม่ลบไฟล์ Excel ต้นฉบับของผู้ใช้ และไม่กระทบ Run อื่น
- Run ที่ลบจะหายจากประวัติทันที พร้อม Cleanup ไฟล์แบบ Retry เมื่อ Excel ปล่อย File Lock
- เพิ่ม Delete Tombstone ป้องกัน Workflow ที่ถูกลบเริ่มขั้นตอนใหม่ภายหลัง
