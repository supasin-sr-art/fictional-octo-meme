BLACKWOLF Web V2.6.5 Run Retention
==================================

1. ปรับชื่อรายการใน Run History
   - จาก Run ID เช่น 20260623-083355-16600
   - เป็น “เช็คกรมธรรม์ต่างด้าวที่ยังไม่ออก 2026-06-23”
   - Run ID ภายในยังคงใช้สำหรับการเปิดผลลัพธ์และลบ Run อย่างถูกต้อง แต่ไม่แสดงเป็นชื่อหลัก

2. เพิ่มระบบเก็บข้อมูล Run 14 วัน
   - คำนวณวันหมดอายุจากเวลาที่เริ่ม Run
   - แสดงตัวนับถอยหลังใน Run History แบบวัน ชั่วโมง และนาที
   - อัปเดตตัวนับถอยหลังอัตโนมัติทุก 1 นาที

3. ลบ Run เก่าอัตโนมัติ
   - ตรวจเมื่อเปิดโปรแกรม
   - ตรวจทุก 30 นาทีขณะโปรแกรมทำงาน
   - ตรวจซ้ำเมื่อเปิดหรือรีเฟรชหน้า Run History
   - ลบเฉพาะประวัติและโฟลเดอร์ Run ภายใน BLACKWOLF ที่ครบ 14 วัน
   - ไม่ลบไฟล์ Excel ต้นฉบับ ไฟล์ที่ดาวน์โหลดแล้ว หรือไฟล์ที่คัดลอกออกไปยังโฟลเดอร์ภายนอก

4. ความปลอดภัย
   - ไม่ลบ Run ที่ยังมี Process ทำงานอยู่
   - ปุ่มลบราย Run และหยุดแล้วลบยังใช้งานได้ตามเดิม
